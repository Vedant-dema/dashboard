## FEATURE-030: Professional Fuhrpark / Kennzeichen List UI
**Date:** 2026-03-27
**Author/Agent:** Cursor AI
**Status:** Modified

### What Was Added
The single-text Kennzeichen input in both the NewCustomerModal (Waschanlage tab) and the CustomersPage detail view was replaced with a professional "Fuhrpark / Kennzeichen" card. Users can now type a plate number, press Enter or click "+", and it appears in a clean vertical list. Each plate has a red trash icon for removal. The Waschanlage tab was also reorganized into three clearly separated cards: "Konto & Rechnung", "Fuhrpark / Kennzeichen", "Bank & Zahlung", and "Wasch-Infos" for a cleaner, more professional layout.

### Where It Was Added

- `frontend/src/pages/CustomersPage.tsx` — replaced single Kennzeichen input in detail drawer with same professional Fuhrpark / Kennzeichen list card; added `newPlateInput` local state

5. In CustomersPage, the existing `draftWash.kennzeichen` string is split on commas for display and re-joined on mutation.
