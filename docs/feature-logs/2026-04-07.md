## FEATURE-144: Customer 360 summary strip — company name + primary address
**Date:** 2026-04-07
**Author/Agent:** Cursor AI
**Status:** Modified

### What Was Added
The edit-mode **Customer 360** band above the tabs now shows **company name** and the **primary address** (first address row: street, ZIP/city, country), driven by live form state so edits are visible before save.

### Where It Was Added
- `frontend/src/components/NewCustomerModal.tsx` — `customer360EditSummary` adds `companyLine` / `addressLine`; strip UI block above chips
- `frontend/src/contexts/LanguageContext.tsx` — `customer360StripCompany`, `customer360StripAddress` (de, en, fr)

### What It Does (Technical)
1. `companyLine` = trimmed `form.firmenname`.
2. `addressLine` = joined `form.adressen[0]` fields: `strasse`, `plz` + `ort`, `land_code`.
3. Empty strings show an em dash in the UI; long text truncates (company) or `line-clamp-2` (address) with `title` for full hover text.

### Data It Accepts / Emits
| Field | Type | Required | Description |
|----|---|----|----|
| form.firmenname | string | No | Display in strip |
| form.adressen[0] | AdresseEntry | No | Primary address line |
