## FEATURE-301: Timetable minimal offer form — layout + Bestandsfluss colors
**Date:** 2026-04-16 
**Author/Agent:** Cursor AI 
**Status:** Modified 

### What Was Added
The **Angebot** column form (**`TimetableOfferMinimalBlock`**) is reorganized into **stacked sections** with **dividers**: vehicle **2×2 grid**, **mileage** on its own band (constrained width on desktop), then **pricing + Bestandsfluss** on one **`lg:grid-cols-3`** row so **Seller / Purchasing / Stock** labels align. The **mutual-exclusion hint** and **conflict** banner sit **full-width above** that row (no longer only beside the buttons). **Bestandsfluss** tiles use **green / red / blue** (purchase / disposal / offer) with matching **icon** tints; **Verkauft** no longer uses amber. **`TimetableOfferModal`** ledger pills use the same **red / blue** active colors for disposal / neutral.

### Where It Was Added
- `frontend/src/pages/timetable/components/TimetableOfferMinimalBlock.tsx` — section layout; **`ledgerTileStyles`**; **`activeLedgerMode`** `null` on **conflict** so no lane appears falsely selected.
- `frontend/src/pages/timetable/TimetableOfferModal.tsx` — active pill classes for **disposal** (**red**) and **neutral** (**blue**).

### What It Does (Technical)
1. Vehicle fields remain **`sm:grid-cols-2`** with consistent **`gap-x-5 gap-y-4`**.
2. Mileage **`max-w`** wrapper avoids a lone full-width field stretching awkwardly.
3. Hint + optional conflict alert, then **`grid gap-5 sm:grid-cols-2 lg:grid-cols-3`**; ledger column **`sm:col-span-2 lg:col-span-1`** so on small screens prices sit row 1–2 and stock row 3 full width.
4. **`ledgerTileStyles`** returns Tailwind **button** + **icon** classes per lane and **`active`** state.
