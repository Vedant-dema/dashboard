## FEATURE-325: Timetable offer tab — Angebotsarchiv as column beside Preisvergleich

## FEATURE-334: Timetable offer — scalable vehicle strip (many trucks)
**Date:** 2026-04-17
**Author/Agent:** Cursor AI
**Status:** Added

### What Was Added
When a contact has **many vehicle offer lines** (e.g. dozens of trucks), the **Vehicles / offers** row no longer tries to show every chip inline. Above a configurable threshold (see **FEATURE-336**: **more than six** lines), the UI shows a **compact trigger** (current selection, total count, chevron) that opens a **searchable, scrollable list** to switch or remove lines. At or below that count, chips stay visible in a **single horizontal scroll row** so the layout stays predictable without wrapping clutter.

### Where It Was Added
- `frontend/src/pages/timetable/components/TimetableOfferVehicleStrip.tsx` — `VEHICLE_STRIP_PICKER_THRESHOLD`, `VehicleOfferOverflowPicker`, conditional layout for strip vs picker; inline chips use `overflow-x-auto` and `truncate` caps.
- `frontend/src/contexts/LanguageContext.tsx` — `timetableOfferVehiclePicker*` keys in `MESSAGES.de` / `en` / `fr` and all nine `LANGUAGE_OVERRIDES` blocks.

### What It Does (Technical)
1. If `offers.length > VEHICLE_STRIP_PICKER_THRESHOLD` (see **FEATURE-336**, currently `6`), render `VehicleOfferOverflowPicker`: button shows ledger-colored shell for the active offer, truncated label, numeric badge, chevron; click toggles panel.
2. Panel: filter `input` (type `search`) filters by concatenated chip label, brand, model, `vehicle_type`, and trailing model digits; list rows call `onSelect` / `onRemove`; outside click and Escape close; query resets when closed.
3. Otherwise render chip UX inside `flex-nowrap overflow-x-auto`; add button remains `shrink-0` beside the scroller on `sm+`.

### Data It Accepts / Emits
| Field | Type | Required | Description |
|----|---|----|----|
| `VEHICLE_STRIP_PICKER_THRESHOLD` | `6` (constant, FEATURE-336) | N/A | Count above which the overflow picker is used instead of inline chips. |
| Picker props | same as strip | Yes | Reuses `offers`, `selectedId`, `onSelect`, `onRemove`, `t`. |

## FEATURE-353: Universal motion tiers — Settings, `data-dema-motion`, Tailwind `dema-low-motion`
